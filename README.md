## Installation
```shell
cd ~/MagicMirror/modules
git clone https://github.com/Prog-Party/MMM-GoogleSheets.git
```

## Configuration
Add MMM-GoogleSheets to ../MagicMirror/config/config.js:

### Simple Version
```javascript
{
  module: "MMM-GoogleSheets",
  position: "top_right",
  config: {
    events: [
      {
      },
    ]
  }
},
```
